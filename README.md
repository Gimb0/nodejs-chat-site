# Node.js Chat Site

Node.js is gaining in popularity so I decided to try it out and made a simple chat site.

## Running the site

Clone the repository
```sh
$ git clone https://github.com/Gimb0/nodejs-chat-site.git
$ cd nodejs-chat-site
```
Install necessary modules
```sh
$ npm install
```

Run the server
```sh
$ node server.js
```
